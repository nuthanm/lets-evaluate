# 📸 UI Screenshots

This folder contains screenshots of each page in the **Let's Evaluate** app.

## How to contribute screenshots

1. Run the app locally: `streamlit run app.py`
2. Visit each page and take a full-page screenshot
3. Save the screenshot using the filename listed below
4. Open a pull request with the added images
